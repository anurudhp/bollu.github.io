§ Parallelisable version of maximum sum subarray

I learnt of a "prefix sum/min" based formulation from the solution to question D, codeforces educational round 88. The idea is to start with the max prefix sum as the difference of right minus left:
max⁡(L,R):∑L≤i≤Ra[i]=max⁡R:(∑0≤i≤Ra[i]−minL:∑0≤i≤L≤R) \begin{aligned} &\max_{(L, R)}: \sum_{L \leq i \leq R} a[i] \\ &= \max_R: \left(\sum_{0 \leq i \leq R} a[i] - min L: \sum_{0 \leq i \leq L \leq R} \right) \\ \end{aligned} ​(L,R)max​:L≤i≤R∑​a[i]=Rmax​:(0≤i≤R∑​a[i]−minL:0≤i≤L≤R∑​)​
Which is then expressed as:
asum[n]≡∑0≤i≤na[i]=max⁡R:(asum[R]−min(L≤R):asum[L]) asum[n] \equiv \sum_{0 \leq i \leq n} a[i] = \max_R: (asum[R] - min_{(L \leq R)}: asum[L]) asum[n]≡0≤i≤n∑​a[i]=Rmax​:(asum[R]−min(L≤R)​:asum[L])
aminsum[n]≡min⁡0≤i≤nasum[i]=max⁡R:(asum[R]−aminsum[R]) aminsum[n] \equiv \min_{0 \leq i \leq n} asum[i] = \max_R: (asum[R] - aminsum[R]) aminsum[n]≡0≤i≤nmin​asum[i]=Rmax​:(asum[R]−aminsum[R])
Since asumasumasum is a prefix-sum of aaa, and aminaminamin is a prefix min of asumasumasum, the whole thing is O(n)O(n)O(n) serial, O(log⁡n)O(\log n)O(logn) parallel. In haskell, this translates to: 
let sums = scanl (+) 0
let sums_mins = scanl1 min . sums
let rise xs = zipWith (-) (sums xs) (sums_mins xs)
main = rise [1, 2, 3, -2, -1, -4, 4, 6]
> [0,1,3,6,4,3,0,4,10]
sums_mins keeps track of the sea level, while the zipWith (-) xxx sums_mins computes the elevation from the sea level.

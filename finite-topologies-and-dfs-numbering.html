§ Finite topologies and DFS numbering

In this great math overflow question on How to think about non hausdorff topologies in the finite case, there's an answer that encourages us to think of them as preorders, which are basically graphs. I wanted to understand this perspective, as well as connect it to DFS numbers, since they provide a nice way to embed these topologies into R\mathbb RR.

§ Closure axioms of topology

We can axiomatize a topology using the kurotawski closure axioms. We need an idempotent monotonic function c:2X→2Xc: 2^X \rightarrow 2^Xc:2X→2X which satisfies some technical conditions. Formally:
  1. c(∅)=∅)c(\emptyset) = \emptyset)c(∅)=∅) [ccc is a strict function: it takes bottoms to bottoms]
  2. A⊆c(A)A \subseteq c(A)A⊆c(A). [monotonicity]
  3. ccc is idempotent: c(c(A))=c(A)c(c(A)) = c(A)c(c(A))=c(A). [idempotence]
  4. for all A,BA, BA,B in XXX, c(A∪B)=c(A)∪c(B)c(A \cup B) = c(A) \cup c(B)c(A∪B)=c(A)∪c(B).
Under this, a set is closed if it is a fixed point of ccc: That is, a set AAA is closed iff c(A)=Ac(A) = Ac(A)=A.

§ Finite topologies as preorders

We draw an arrow x→yx \rightarrow yx→y iff x∈Closure(y)x \in Closure(y)x∈Closure(y). Alternatively stated, draw an arrow iff Closure(x)⊆Closure(y)Closure(x) \subseteq Closure(y)Closure(x)⊆Closure(y). That is, we have an injection from the closure of xxx into the closure of yyy, and the arrow represents the injection. Alternatively, we can think of this as ordering the elements xxx by "information". A point xxx has less information than point yyy if its closure has fewer points.
